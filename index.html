About.

Hexa is an animation library, simple, manageable and awesome. you can animate an element just by adding an animation class

Installation

a - Using Git

$ git clone https://github.com/amine1107/Hexa.git

b - With bower

bower install hexa-animation

c - With npm

npm install hexa-animation

Basic Usage

you can simply adding Hexa to your html file

<link rel="stylesheet" type="text/css" href="hexa.css">

Add Class

<div class="hexa jump">
